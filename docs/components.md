# Components Catalog

The UI Kit includes the following reusable components. Each component is theme‑aware, built on Angular Material, and has associated Storybook stories.

| Component  | Description                                              | Storybook Link                                           |
| ---------- | -------------------------------------------------------- | -------------------------------------------------------- |
| Button     | Primary action button with variants and loading state.   | `http://localhost:6006/?path=/story/button--primary`     |
| Card       | Container with header, content, and actions.             | `http://localhost:6006/?path=/story/card--default`       |
| Checkbox   | Styled checkbox with indeterminate state.                | `http://localhost:6006/?path=/story/checkbox--basic`     |
| Form Field | Wrapper for inputs with label, hint, and error display.  | `http://localhost:6006/?path=/story/form-field--default` |
| Table      | Data table with sorting, pagination, and actions column. | `http://localhost:6006/?path=/story/table--default`      |
| Textarea   | Multi‑line text input with auto‑resize.                  | `http://localhost:6006/?path=/story/textarea--default`   |
| Uploader   | File upload component with drag & drop support.          | `http://localhost:6006/?path=/story/uploader--default`   |

> **Note**: Storybook must be running (`npx nx storybook ui-kit`) for the links to work.
